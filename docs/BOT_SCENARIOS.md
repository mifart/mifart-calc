# BOT_SCENARIOS — Сценарии чат-ботов MIFART

## Общий тон общения
- Дружелюбный, но профессиональный
- Говорим «вы» (с маленькой буквы)
- Без канцеляризмов
- Конкретные цифры и факты
- Ответ в течение 15 минут

### Сценарий «💰 Рассчитать стоимость»

**Шаг 1:**
```
Что хотите построить?
```
_Inline кнопки:_ 🏠 Дом | 🛁 Баня | 🌳 Беседка | 📐 Другое

**Шаг 2:**
```
Отлично! Укажите примерные размеры (например: 6x8 или 50 кв.м.)
Или просто напишите площадь, которую хотите:
```

**Шаг 3:**
```
В каком городе / регионе планируете строить?
```

**Шаг 4:**
```
Нужен ли фундамент? (если участок уже есть — просто напишите "нет")
```
_Кнопки:_ Да, нужен | Нет, есть готовый | Не знаю

**Шаг 5:**
```
Есть ли примерный бюджет? Это поможет подобрать нужную комплектацию.
```
_Кнопки:_ До 500 тыс | 500–1 млн | 1–2 млн | 2+ млн | Пока не знаю

**Шаг 6:**
```
Почти готово! Оставьте ваш номер телефона — менеджер перезвонит в течение 15 минут и подготовит точный расчёт.
```

**Шаг 7 (после получения телефона):**
```
Есть комментарий или пожелание? Или напишите "нет", если всё понятно.
```

**Финал:**
```
✅ Заявка принята!

Наш менеджер перезвонит вам в течение 15 минут и ответит на все вопросы.

📞 Телефон: +7 (995) 380-08-88
🌐 Сайт: mifart.ru

Спасибо, что выбрали МИФАРТ! 🏗
```

### Anti-spam защита
- Flood control: не более 1 заявки с одного chat_id за 24 часа
- Blacklist: блокировка спам-паттернов в телефоне
