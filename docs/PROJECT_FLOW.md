# PROJECT_FLOW — Карта пути клиента MIFART

## Главная цель
Пользователь → Заявка → Менеджер → Сделка

---

## Источники трафика (откуда приходит пользователь)

| Источник | Канал | UTM_source |
|---|---|---|
| Яндекс / Google | Органика | yandex / google |
| Яндекс.Директ | Платная реклама | yandex | direct |
| VK Реклама | Таргет | vk |
| Telegram-канал MIFART | Органика | telegram |
| MAX канал MIFART | Органика | max |
| Прямой переход | Прямой | direct |

---

## Путь 1 — Через Сайт

```
Реклама / Поиск / Ссылка
        ↓
Главная страница mifart.ru
        ↓
Hero: видит заголовок + 4 кнопки
        ↓
[Рассчитать стоимость] → Скролл до Calculator
        ↓
Калькулятор: выбирает тип (Дом/Баня/Беседка) + площадь
        ↓
Видит ориентировочную цену
        ↓
Форма: Имя + Телефон → [Получить расчёт]
        ↓
POST /api/leads
        ↓
├── Telegram уведомление менеджеру
├── Bitrix24 — создаётся Лид
└── n8n webhook (опционально)
        ↓
Сообщение «Заявка принята! Перезвоним за 15 минут»
```

**Альтернативный путь:**
```
Страница услуги (Дома / Бани / Беседки)
        ↓
CTA «Рассчитать стоимость»
        ↓
Форма → /api/leads → Менеджер
```

## Путь 2 — Через Telegram-бот

```
Ссылка на бота / QR-код / Кнопка на сайте
        ↓
/start — Приветствие + меню
        ↓
Выбирает услугу из кнопок
        ↓
Получает описание / фото
        ↓
[💰 Рассчитать стоимость]
        ↓
Диалог: тип → размеры → регион → бюджет → телефон
        ↓
POST /api/leads (source_platform: telegram)
        ↓
Уведомление менеджеру
        ↓
Пользователю: «Спасибо! Перезвоним за 15 мин»
```

---

## Путь 5 — Через мини-приложение

```
Кнопка на сайте / кнопка в боте / ссылка
        ↓
Мини-приложение открывается
        ↓
Шаг 1: Выбор типа объекта
Шаг 2: Ввод параметров (длина, ширина, высота, класс)
Шаг 3: Показ ориентировочной цены
Шаг 4: Форма (Имя + Телефон)
        ↓
POST /api/leads (source_platform: miniapp)
        ↓
Уведомление менеджеру
```

## Единая структура заявки (LeadData)

```typescript
{
  source_platform: 'website' | 'telegram' | 'max' | 'vk' | 'miniapp',
  source_page: string,           // URL или название экрана
  user_name: string,
  phone: string,
  service_type: string,          // 'house' | 'bath' | 'gazebo' | 'custom'
  object_type?: string,
  dimensions?: string,           // '6x8' или '50 кв.м.'
  city?: string,
  region?: string,
  budget?: string,
  comment?: string,
  utm_source?: string,
  utm_medium?: string,
  utm_campaign?: string,
  utm_content?: string,
  utm_term?: string,
  created_at: string,            // ISO 8601
}
```

## Куда уходят лиды

| Получатель | Метод | Условие |
|---|---|---|
| Менеджер (Telegram) | Bot API sendMessage | Всегда |
| Bitrix24 | REST Webhook (crm.lead.add) | Если задан BITRIX24_WEBHOOK_URL |
| n8n | HTTP POST | Если задан N8N_WEBHOOK_URL |
| Консоль / Лог | console.log | Всегда (для дебага) |
